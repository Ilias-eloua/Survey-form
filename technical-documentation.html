Intro to HTML Forms

HTML forms are used to collect user data and send it to a server. They consist of various form elements that allow users to input information, such as text, choices, buttons, and more.

An HTML form is created using the <form> element. The form contains different input elements like <input>, <textarea>, <select>, and <button> that users interact with.

Basic structure of an HTML form:


<form action="submit_form.php" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  <button type="submit">Submit</button>
</form>

In this example:

  • The form has an action that determines where the data will be sent.
  • The method specifies how the data is transmitted (POST or GET).
  • The input elements (<input>) include a type to indicate what kind of data is being entered, such as text or email.

HTML Form Elements

HTML offers various elements for user input. Below are the most commonly used ones:

1. Text Input

<input type="text" name="username" placeholder="Enter your name">

A basic text field for short input.

2. Email Input

<input type="email" name="email" placeholder="Enter your email">

Automatically validates if the input is a valid email address.

3. Password Input

<input type="password" name="password">

Masks the input with dots or asterisks.

4. Textarea

<textarea name="message" rows="4" cols="30"></textarea>

Used for longer text input such as messages or comments.

5. Checkbox

<label><input type="checkbox" name="newsletter"> Subscribe to newsletter</label>

Lets users select or deselect an option.

6. Radio Buttons

<label><input type="radio" name="color" value="red"> Red</label>
<label><input type="radio" name="color" value="blue"> Blue</label>

Allows users to select only one option from a set.

7. Select Dropdown

<select name="language">
  <option value="en">English</option>
  <option value="nl">Dutch</option>
</select>

Used to create a dropdown menu with multiple options.

Basic CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It helps control the appearance of HTML elements, such as text, images, and layout.

1. Changing Text Color

h1 { color: blue;}

This changes the text color of all <h1> elements to blue, making the heading stand out more.

2. Setting Background Color

body { background-color: lightgray;}

This sets the background of the whole page to light gray, giving the page a neutral tone.

3. Changing Font Size

p {font-size: 16px;}

This sets the font size of all paragraphs to 16 pixels, ensuring readability for the text.

4. Adding Borders

img {border: 2px solid black;}

This adds a solid black border around images, which can help make them visually distinct on the page.

5. Adding Padding

div {padding: 10px;}

This adds space inside the <div> element, around its content, preventing it from touching the edges of the element.

  • CSS controls the look and feel of the website, making it more visually appealing.
  • CSS uses selectors to target HTML elements and apply styles to them.
  • Values like px, %, and em control size, spacing, and layout.
  • CSS can be written inline within an HTML element, in a <style> tag within the document head, or in an external stylesheet.
  • Basic properties include color, background-color, font-size, border, margin, and padding.

CSS Colors

CSS allows you to add color to your webpage elements. There are several ways to define colors in CSS:

1. Named Colors

You can use common color names like red, blue, green, etc.

h1 { color: red; }

This will change the text color of the <h1> element to red.

2. Hexadecimal Colors

Hex codes start with # followed by 6 characters, usually representing RGB values.

p { color: #00FF00; }

This applies a green color to the text, using its hexadecimal representation.

3. RGB Colors

Use rgb() to define red, green, and blue values, where each value ranges from 0 to 255.

div { background-color: rgb(255, 0, 0); }

This sets the background color of the <div> element to red using the RGB model.

4. HSL Colors

hsl() uses hue, saturation, and lightness to define colors. Hue is the color itself, saturation determines the intensity, and lightness sets the brightness.

span { color: hsl(240, 100%, 50%); }

This sets the color to a shade of blue with full saturation and medium lightness.

5. Opacity with RGBA

Add transparency using rgba(), which includes an alpha value for opacity, where 1 is fully opaque and 0 is fully transparent.

.box { background-color: rgba(0, 0, 255, 0.5); }

This creates a semi-transparent blue background for the <div> with an opacity of 50%.

  • color sets the text color.
  • background-color sets the background color of elements.
  • CSS supports color definitions using names, hex, rgb, hsl, and rgba formats.

CSS Box Model

In CSS, every element is represented as a box — this is called the box model. It describes how the width and height of elements are calculated.

The box model consists of four parts: content, padding, border, and margin.

  • Content: the actual content (text or images).
  • Padding: space between the content and the border.
  • Border: the edge around the element.
  • Margin: space between this element and others.

Understanding the box model helps you control exactly how an element appears on the page.

div { 
  width: 200px;
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}

The total width is: 200 + 10*2 (padding) + 2*2 (border) = 224px.

Use box-sizing: border-box to include padding and border within the specified width.

* {
  box-sizing: border-box;
}

CSS Flexbox

Flexbox is a CSS layout model that makes it easier to design flexible and responsive layout structures.

It allows elements within a container to automatically adjust based on screen size and available space.

To use Flexbox, you set the container's display property to flex.

You can then use properties like justify-content and align-items to control alignment.

Flexbox is useful for both horizontal and vertical layouts.

  • display: flex; – activates flex context.
  • justify-content – aligns items horizontally.
  • align-items – aligns items vertically.
  • flex-direction – sets the direction (row, column).
  • flex-wrap – allows items to wrap onto multiple lines.
display: flex;
justify-content: center;
align-items: center;
flex-direction: row;
flex-wrap: wrap;

Typography

Typography in CSS controls how text appears on a webpage, including font style, size, spacing, and alignment.

Good typography improves readability and user experience.

You can set the font using the font-family property.

The size of the text is adjusted with font-size.

Other properties like line-height and letter-spacing fine-tune text appearance.

  • font-family – sets the typeface.
  • font-size – controls text size.
  • line-height – controls space between lines.
  • letter-spacing – controls space between letters.
  • text-align – aligns the text (left, center, right).
font-family: Arial, sans-serif;
font-size: 16px;
line-height: 1.5;
letter-spacing: 0.5px;
text-align: center;

Accessibility

Accessibility makes websites usable for everyone, including people with disabilities who rely on screen readers or keyboard navigation.

Semantic HTML elements help assistive technologies interpret content correctly.

Tags like <label>, <button>, and <nav> improve accessibility and structure.

Ensure all interactive elements are keyboard-accessible and that focus indicators are visible.

Always use descriptive alt text for images to convey their purpose or meaning.

  • Use semantic tags like <header>, <main>, and <footer>
  • Link <label> to form inputs using the for attribute
  • Make sure focus is clearly visible when tabbing
  • Use meaningful link text (e.g., "Read more about pricing" instead of "Click here")
  • Add captions or transcripts for audio and video content

                  <img src="photo.jpg" alt="Smiling cat">

                  <nav>...</nav>

Pseudo selectors

Pseudo-selectors let you style elements based on their state or position, even if they don’t have a class or ID.

They start with a colon (:) and can be combined with regular selectors.

Common pseudo-selectors include :hover, :focus, :first-child, and :nth-child().

They are useful for interactivity and more precise targeting in CSS.

You can style form elements, lists, or any element differently depending on user interaction.

  • :hover — when the mouse is over an element
  • :focus — when an element (like an input) is selected
  • :first-child — targets the first child of a parent
  • :nth-child(2) — targets the second child
  • :active — when an element is being clicked
button:hover { background-color: blue; }
                  input:focus { border-color: green; }
                  li:first-child { font-weight: bold; }

                  p:nth-child(2) { color: gray; }

                  a:active { color: red; }

Intermediate CSS

Intermediate CSS concepts dive deeper into styling techniques that help create responsive and well-structured websites.

Common intermediate concepts include CSS Grid, CSS Flexbox, transitions, and animations.

These techniques allow you to create more dynamic layouts and interactive user interfaces.

Mastering these concepts will help you build modern web applications.

  • CSS Grid — allows you to create complex two-dimensional layouts
  • CSS Flexbox — used for creating flexible, one-dimensional layouts
  • Transitions — enable smooth changes in style properties
  • Animations — used for creating keyframe-based animations in CSS
  • Media Queries — to adapt your website to different screen sizes
.container { display: grid; grid-template-columns: repeat(3, 1fr); }
                  .flex-container { display: flex; justify-content: center; }
                  
                    button { transition: background-color 0.3s ease; }
                   @keyframes move { 0% { transform: translateX(0); } 100% { transform: translateX(100px); } }
                  
                    @media (max-width: 600px) { .container { grid-template-columns: 1fr; } }

Responsive Web Design

Responsive web design (RWD) is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes.

It ensures that websites work across different devices such as desktops, tablets, and mobile phones.

Key concepts in responsive design include fluid grids, flexible images, and media queries.

  • Fluid Grids — use percentage-based widths to create flexible layouts
  • Flexible Images — images that scale based on the viewport size
  • Media Queries — CSS rules that apply based on device characteristics like screen width
  • Viewport Units — use of units like vw, vh for responsive design
  • Mobile-first Approach — designing for smaller screens first and then scaling up
body { max-width: 100%; margin: 0; }

                  img { max-width: 100%; height: auto; }
                

                  @media (max-width: 768px) { .container { flex-direction: column; } }

                html { font-size: 2vw; }

                @media (min-width: 600px) { .container { display: grid; grid-template-columns: 1fr 1fr; } }
